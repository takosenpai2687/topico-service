## Topico Web Service

### Modules

- __admin__: admin related, e.g., delete community, post, comment ban and unban user.
- __common__: entities, utils, enums, dtos, basic types, etc.
- __eureka__: service discovery server.
- __explore__: search, trending, recommendation.
- __gateway__: api gateway for load balancing, authentication and routing.
- __home__: crud apis for home page, community detail, post detail, comment.
- __notification__: notification related apis.
- __user__: user related apis: edit profile, reset password, etc.

## Tech Stacks

- Java 20
- Spring Cloud
- Spring Boot 3.1.2
- Spring Data JPA
- MySQL
- Redis
- Kafka
